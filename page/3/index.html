遍历文件夹处理

此外还可以搭配xargs遍历整个目录

  • hadoop -ls -h hdfs://your_path 列出整个目录
  • awk -F' ' '{print $NF}' 取出最后一列，也就是文件地址列
  • awk -F'/' '{print $NF}' 将地址中的文件名取出
  • xargs -I {} bash -c ... 将每个文件名依次作为参数处理
  • "hadoop -cat ..." 按照之前流式处理的方式处理每一个文件
