谨慎使用CREATE-SELECT语句复制表数据

发表于 2022-08-03 分类于 backend 阅读次数：
本文字数： 248 阅读时长 ≈ 1 分钟

在复制表时，可以使用CREATE-SELECT来同时复制表结构和数据

但这样做实际上是有问题的

原因在于，CREATE-SELECT在创建新表时仅复制了字段类型，但是不会设置主键子增等属性。

因此，在复制表时，最好还是使用CREATE-LIKE和INSERT来复制
